# Utility Hub — Python

A curated collection of small Python applications and practical command-line tools.

## Included projects

- `file-organizer/` — sorts files into categories, with a companion batch launcher.
- `qr-code-attendance-scanner/` — desktop QR-code attendance scanner that records time-in and time-out data.

Each project keeps its own README and can be used independently.

## Repository history

`file-organizer` and `qr-code-attendance-scanner` were consolidated here from their former standalone repositories in July 2026. The original repositories are archived for reference.
